# Python - More Classes and Objects

* what does ```@property```do?
The @property lets a method to be accesded as an attribute instead of as a method with ```()```

* When to use @property
@property is used when an attribute is derived from other attribute in the class, so the derived attribute will update whenever the source attrubute is changed.
* Setter is used if you want to update the source attributes/variable if the property is set.
